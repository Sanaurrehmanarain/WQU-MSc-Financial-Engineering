# Master of Science in Financial Engineering (MScFE)

**WorldQuant University (WQU)**

This repository contains my complete academic coursework and projects from the **MSc in Financial Engineering (MScFE)** program at WorldQuant University. It serves as a personal archive and a portfolio of my studies in quantitative finance, machine learning, and algorithmic trading.

🏛️ **About WorldQuant University**
WorldQuant University is a global, not-for-profit educational institution offering a tuition-free, globally accessible Master's degree in Financial Engineering. The program focuses on the intersection of finance, mathematics, and computer science, training students to analyze financial markets and build sophisticated quantitative models.

📂 **Repository Structure**
The repository is organized chronologically by course. Each course folder typically contains:
* **Lessons:** Jupyter Notebooks (`.ipynb`) and required datasets for weekly modules.
* **Projects:** Applied group or individual projects containing code, datasets, and final project reports.

### 📘 Courses Included

* 📁 **1. Financial Markets**
* 📁 **2. Financial Data**
* 📁 **3. Financial Econometrics**
* 📁 **4. Derivative Pricing**
* 📁 **5. Stochastic Modeling**
* 📁 **6. Machine Learning in Finance**
* 📁 **7. Deep Learning for Finance**
* 📁 **8. Portfolio Management**
* 📁 **9. Risk Management**

🎯 **Program Highlights**
* **Quantitative Modeling:** Stochastic calculus, derivative pricing, and econometrics.
* **Data Science & AI:** Implementation of Machine Learning and Deep Learning models for financial time-series forecasting and pattern recognition.
* **Tools:** Heavy emphasis on Python, Pandas, NumPy, Scikit-Learn, TensorFlow/PyTorch, and Jupyter environments.
